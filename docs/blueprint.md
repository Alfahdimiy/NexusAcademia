# **App Name**: NexusAcademia

## Core Features:

- User Management: Admin, Instructor, and Student roles with distinct permissions and profiles.
- Course Management: Create, organize, and manage courses with Units/Modules.
- Enrollment System: Manage student enrollments in courses.
- Weekly Learning Path: Structured learning paths to organize course content.
- Discussion Forum: Peer interaction, requiring initial responses before seeing other posts.
- Assignment Upload: Secure system for uploading and managing assignments.
- Peer Assessment: System to randomly assign papers for grading based on a rubric.
- Learning Journal: Communication channel between student and instructor for weekly reflections.
- AI Tutoring Assistant: Integrate an AI-powered assistant to act as a tool for summarizing lectures and creating simple quizzes from submitted lecture notes.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to evoke a sense of knowledge and sophistication.
- Background color: Light grey (#F5F5F5), a desaturated version of purple to create a clean and focused backdrop.
- Accent color: Electric blue (#3F51B5), an analogous color that adds contrast and highlights interactive elements.
- Headline font: 'Space Grotesk', sans-serif, for a modern tech-forward feel. Body font: 'Inter', sans-serif, for readability.
- Use flat, consistent icons with subtle animations to represent course topics and actions.
- Implement a clean, responsive layout with clear hierarchy for intuitive navigation.
- Subtle transitions and feedback animations to enhance user experience.
